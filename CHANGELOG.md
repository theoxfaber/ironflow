# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-05-09

### Added
- Initial release of Ironflow.
- DAG scheduling with cron support.
- Bash, Python, HTTP, SQL, and Slack operators.
- XCom for cross-task data passing.
- Embedded SQLite persistence with WAL mode.
- Crash recovery system for orphaned runs.
- REST API for DAG management.
- Embedded Web UI for monitoring and triggering DAGs.
- CLI for interacting with the orchestrator.
- Comprehensive benchmark suite comparing with Airflow, Dagster, and Prefect.
